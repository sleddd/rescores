The Hybrid WordPress Theme

One part block theme, one part classic theme, Rescores hybrid WordPress theme lets you take advantage of both the old and new way of building WordPress sites with support for the full site editor and the customizer.

Photo by Josep Martins on Unsplash

Learn More

Exploring the line between blocks and classic development, Rescores is an experimental theme to help you develop new WordPress websites.

Build how you like, then zip it up

You can use theme.json to style theme blocks, while the integrated Tailwind workflow transpiles theme.json into global css which can be used for non-block elements like headings, links, and paragraphs. Once you are done coding, use npm run run zip to zip up your theme for uploading to a new site leaving the development tools out of production code.

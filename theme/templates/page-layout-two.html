Re-examining how you build

Building a block theme will get you part of the way there and in many cases all the way there for smaller sites, however, with larger blogs that have custom features, you may want more advanced patterns or templates requiring PHP. The WP theme.json lets you style many common elements and blocks, but anything that falls outside of that is not included unless you create custom styles. This could includes things like tables, lists, code, and pre elements. Rescores lets you use Tailwind to add in the extra css you might need without all the bulk of a full css framework. You can also add elements to the theme.json and they will be transpiled via Tailwind into global css. When your all done building, just use npm run zip to create a zip file of your theme that excludes your development tools and is ready to upload and activate.
